# AIP-2: Cross-Chain Asset Bridge

Author: Arta Blockchain Team  
Status: Draft  
Type: Protocol  
Created: 2026

## Abstract

This proposal introduces a cross-chain asset bridge for the Arta Blockchain ecosystem, enabling tokens to move between supported blockchain networks.

## Motivation

Arta Blockchain is designed as a multi-chain infrastructure supporting assets across different blockchain ecosystems.

## Specification

Supported networks:

- Ethereum
- BNB Chain
- Arbitrum
- Solana

Supported assets:

- USDT.a
- USDC.a
- ART ecosystem tokens

## Security

The bridge will use a multi-signature validator architecture to ensure secure asset transfers between chains.
