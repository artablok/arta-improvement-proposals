# Contributing to Arta Blockchain

Thank you for your interest in contributing to the Arta Blockchain ecosystem.

## How to contribute

You can contribute by:

• submitting Arta Improvement Proposals (AIPs)
• reporting issues
• suggesting protocol improvements
• improving documentation

## Proposal process

1. Fork the repository
2. Use the AIP-template.md
3. Submit a Pull Request
4. Participate in community discussion

All proposals are reviewed by the Arta Blockchain core contributors.
